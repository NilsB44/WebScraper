## Reporting a Vulnerability

We take the security of this project seriously.

If you find a vulnerability, please **DO NOT** open a public issue. Instead, please report it privately:

1.  Email us at [your-email@example.com](mailto:your-email@example.com).
2.  Or use the GitHub Security Advisory "Report a vulnerability" feature if enabled.

We will try to review and respond to your report within 48 hours.
